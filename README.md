# 📧 Phishing Email Detection

A simple machine learning project that detects whether an email is **Phishing** or **Safe** using text classification.

## 🚀 Features

* Classifies emails as Phishing (1) or Safe (0)
* Uses TF-IDF for text processing
* Trained with Naive Bayes algorithm
* Displays accuracy and confusion matrix

## 🛠️ Tech Stack

* Python
* Scikit-learn
* Pandas
* Matplotlib & Seaborn

## ▶️ Run the Project

```bash
pip install pandas scikit-learn numpy matplotlib seaborn
python main.py
```

## 📊 Output

* Accuracy ~94%
* Confusion Matrix
* Example Prediction: **Phishing / Safe**

## 📂 Files

* `main.py` – model code
* `emails.csv` – dataset
